===========
Quick Start
===========

Installation
============

Please install sphinxcontrib-ogp with using pip (8.1.1 or later).

.. code-block:: bash

   $ pip install sphinxcontrib-ogp


Sphinx conf.py
==============

.. code-block:: python
   :caption: conf.py

   extensions = [
       'sphinxcontrib_ogp',
   ]

   og_site_url = 'http://sphinx-users.jp/'
   og_twitter_site = '@sphinxjp'
